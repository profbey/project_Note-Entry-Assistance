 Python-based teacher's assistant application for managing and analyzing student grades with predictive features.

Core Features:
- Student information entry (name, surname) with predictive suggestions
- Flexible exam grade management that handles missing grades (null values)
- Letter grade conversion system
- Export functionality for student data and grades
- Statistical analysis of grade distribution with explanatory text and visualizations
- Local file storage in the program's directory
- Modern graphical interface using Tkinter

UI/Style:
- Academic dashboard theme with clean typography and structured layouts
- Information cards with clear visual hierarchy for student profiles
- Professional color palette with emphasis on readability and data visualization
- Intuitive navigation between different sections (grade entry, analysis, reports)
